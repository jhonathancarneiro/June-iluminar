📋 Prompt para a IDE
Contexto: Preciso de uma aplicação web simples para hospedar ou vercel ou githubpages, para funcionar como caixa de uma Festa Junina. O destino dos pagamentos será o Banco Inter atraves de um qr code fixo.

Requisitos do Sistema:

Interface de Vendas (Painel):

Botões grandes e fáceis de clicar para adicionar produtos ao carrinho.

Produtos : estao em @cardapio

Exibição do valor total somado em tempo real.

Botão "Limpar Conta" para zerar o total e reiniciar o atendimento.

Fluxo de Pagamento Pix: 

Botão pagamentos, o app nao faz o pagamento.
Se eu escolher pix ele mostra esse qr  estatico e eu falarei o valor para pessoa @pix
Se escolher debito ou credito eu finalizo manualmente na maquininha.


Todas vendas precisaram ser ter log para eu ver depois oque foi vendido de cada coisa e poder entender como foi as vendas de cada item. o armazenamento fica para a forma que for mais facil e rapida.

O sistema nao precisa ser robusto nem escalavel, é uma aplicaçao sazonal e so precisa funcionar e ser criada rapida.



Design/Estilo:

Visual limpo, responsivo (focado em uso notablet) e com temática festiva/junina discreta (cores quentes como laranja, amarelo ou tons amadeirados).
